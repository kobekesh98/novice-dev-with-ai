# Student Projects

This folder contains every student's work for the course.

## Structure

Each student has their own subfolder named after them:

```
student-projects/
└── your-name/
    ├── index.html      ← Your main page
    ├── style.css       ← Your styles
    ├── images/         ← Your images
    └── README.md       ← Brief description of what you built
```

## Rules

- Work **only** inside your own folder
- Never edit another student's folder
- Keep all your files inside `student-projects/your-name/`
- See [`CONTRIBUTING.md`](./CONTRIBUTING.md) for how to submit your work via Pull Request

## Creating Your Folder

On Day 1 during the Zoom setup session, run:

```bash
mkdir -p student-projects/your-name
```

Then create `index.html` and `style.css` inside it.
